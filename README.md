#  Expense Tracker

A full-stack web application designed to help users track and manage their daily expenses efficiently with a clean and simple interface.

---

##  Features
- Add new expenses with details
- View all recorded expenses
- Edit and delete existing entries
- Organized and user-friendly interface
- Backend integration for data handling

---

##  Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- express
-  MongoDB

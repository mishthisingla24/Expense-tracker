# 💸 Expense Tracker

A full-stack web application that helps users track, manage, and analyze their daily expenses with a clean and intuitive interface.

---

## 🚀 Features
- Add, edit, and delete expenses
- View all transactions in an organized way
- Simple and user-friendly interface
- Backend integration for data management
- Modular and scalable code structure

---

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Express.js 

### Database
-  MongoDB

---

## 📂 Project Structure

```
Expense-tracker/
│── frontend/     # Contains UI and client-side logic
│── backend/      # Contains server-side logic and APIs
```

---

## ▶️ How to Run the Project

### 1. Clone the repository
```
git clone https://github.com/mishthisingla24/Expense-tracker.git
```

### 2. Run Frontend
- Open the `frontend` folder  
- Open `index.html` in your browser  

### 3. Run Backend
```
cd backend
npm install
npm start
```
